Reads relaxation data. \mbox{\hyperlink{structRelaxation}{Relaxation}} data should be set out with two components.~\newline
The file should begin with the keys \textquotesingle{}F\+I\+E\+LD\textquotesingle{}, \textquotesingle{}WR\textquotesingle{}, \textquotesingle{}W1\textquotesingle{}, \textquotesingle{}T\+Y\+PE\textquotesingle{} denoting the form of the data, eg~\newline
\textbackslash{}t F\+I\+E\+LD = 700~\newline
\textbackslash{}t WR = 50000~\newline
\textbackslash{}t W1 = 0~\newline
\textbackslash{}t T\+E\+MP = 300~\newline
\textbackslash{}t T\+Y\+PE = 15N\+R1~\newline
Each parameter will default to -\/1 so if one is not set this may cause errors (though a warning will be given). ~\newline
T\+Y\+PE may be one of \mbox{[}15N\+R1, 15N\+R1p, 13C\+R1, 13C\+R1p\mbox{]} currently, though it should be noted that 13C has not been tested.~\newline
\begin{DoxyWarning}{Warning}
There must be a space on each size of the equals sign (eg, T\+E\+MP = 300 is good, T\+E\+MP=300 is bad)~\newline
Once this head section is complete, it should be followed by \char`\"{}\#\+D\+A\+T\+A\char`\"{}.~\newline
After this, data should be split into three columns;~\newline
\textbackslash{}t Column 1\+: residue number (starting from 1)~\newline
\textbackslash{}t Column 2\+: \mbox{\hyperlink{structRelaxation}{Relaxation}} rate (in s-\/1)~\newline
\textbackslash{}t Column 3\+: \mbox{\hyperlink{structRelaxation}{Relaxation}} error, two standard deviations (in s-\/1)~\newline
